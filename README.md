# eris2

A DIY planet + twitter aggregator.

## Prerequisists

* Ruby 1.9
* tidy 
* imagemagick
* PostgreSQL

## Install

    https://github.com/danchoi/eris2
    bundle install
    createdb eris
    psql -d eris -f schema.sql
